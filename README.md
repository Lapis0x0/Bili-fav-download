# 项目介绍

这是一个用于下载B站单个视频/收藏夹全部视频的工具

# 功能

1. 下载单个视频/音频
2. 下载收藏夹中的所有视频/音频

# 使用方法

1. 在.env文件中填入SESSDATA，SESSDATA的获取方法见[这里](https://www.bilibili.com/read/cv25451423/)

2. 安装依赖
   
```bash
pip install -r requirements.txt
```

1. 运行程序
```bash
python main.py
```
